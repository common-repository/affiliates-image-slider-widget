== Description ==
Are you an Affiliates?? Do you want to show your affiliate product in widget?? 
Yes, you are right place now. I just create an Image Slider for you. Affiliates Image Slider Widget working from your Media Library in a Modern Responsive Slider with custom web address which you provide image caption. You can also added ALT and Description. The widget uses universal widget styling, so should fit perfectly into any WordPress theme.


1. Display Multi Images on Widget Areas.
2. Customizable Responsive Slider.
3. Upload and Select Images from Media Library
4. Add your custom address in caption
5. Sort Images by Order
6. Automatic Image Slide or Fade Effects
7. Suitable for Photographers and Graphic Designers
8. Show Image Titles, and Descriptions

For documentation, please visit (https://techaid24.com/boost-your-affiliates-with-affiliates-image-slider-widget/)

== Installation ==
1. Upload the 'affiliates-image-slider-widget' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance > Widgets. Locate the widget called Affiliates Image Slider Widget.
